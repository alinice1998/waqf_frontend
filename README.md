# Waqf Frontend - منصة التقسيم بالوقفات

الواجهة الأمامية لمشروع **وقف**، وهي منصة ويب تفاعلية تتيح للمستخدمين تقسيم التلاوات القرآنية بشكل آلي باستخدام تقنيات الذكاء الاصطناعي، مع توفير بيئة متكاملة لتعديل ومراجعة التوقيتات.

## الميزات الأساسية
- **واجهة مستخدم احترافية:** تصميم عصري وداكن (Dark Mode) مريح للعين مع استخدام Tailwind CSS.
- **عرض التلاوة الصوتي:** دمج مشغل صوتي مرئي باستخدام `Wavesurfer.js` لمتابعة موجات الصوت والكلمات أثناء التلاوة.
- **محرر توقيتات دقيق:** إمكانية تعديل توقيتات الكلمات (بداية/نهاية) بدقة تصل إلى أجزاء من الثانية.
- **تصدير النتائج:** تصدير التوقيتات بصيغ متعددة (JSON, SRT, VTT) لاستخدامها في تطبيقات القرآن الكريم أو الفيديوهات.
- **إعدادات متقدمة:** التحكم بحساسية الصمت، واختيار محرك الوقف (Librosa/Silero)، وتعديل النص القرآني المرجعي.

## التقنيات المستخدمة
- HTML5, CSS3, JavaScript (Vanilla)
- [Tailwind CSS](https://tailwindcss.com/) للتصميم المتجاوب والمكونات الجاهزة.
- [Wavesurfer.js](https://wavesurfer-js.org/) لرسم الموجات الصوتية بشكل تفاعلي.
- [Lucide Icons](https://lucide.dev/) للأيقونات الحديثة.

## كيفية الاستخدام
1. قم بفتح ملف `index.html` في أي متصفح حديث.
2. من القائمة الجانبية، انقر على **أكواد التشغيل (Colab)** وانسخ الأكواد لتشغيل محرك الذكاء الاصطناعي السحابي.
3. افتح Google Colab، الصق الأكواد وشغلها، ثم انسخ **رابط خادم كولاب** الناتج.
4. الصق الرابط في الحقل المخصص بالمنصة.
5. اختر السورة المطلوبة وارفع ملف التلاوة الصوتي (MP3, WAV).
6. انقر على **بدء التقسيم** وانتظر النتيجة لتعديلها أو تصديرها.

## الاتصال بالخادم
الواجهة الأمامية مصممة للاتصال بخادم `Waqf Backend`. في حال التشغيل السحابي من خلال Google Colab، يتم استخدام رابط Cloudflare Tunnel أو أنفاق مشابهة لتسهيل الاتصال بأمان وبدون إعدادات شبكة معقدة.
